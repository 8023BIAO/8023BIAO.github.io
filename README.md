# biao.chat - 个人极简工具箱

> "Simplicity is the ultimate sophistication."
>
> 极简是终极的复杂。

👋 欢迎来到 biao.chat 的开源仓库。这是一个基于原生技术栈构建的个人静态网站，专注于提供实用的小工具，并探索极简主义的网页设计美学。

## ✨ 项目亮点 (Highlights)

这个项目不仅仅是工具的堆砌，更是一次对现代网页交互的探索：

* **🎨 极简美学**：摒弃多余的装饰，利用负空间（留白）、柔和色调和圆角设计，提供舒适的视觉体验。

* **🌗 自动暗色模式**：支持明亮/暗黑主题切换。基于 `localStorage` 和 `pageshow` 事件，实现多页面、跨标签页的**毫秒级状态同步**。

* **📱 响应式设计**：从 4K 显示器到手机屏幕，布局自动流转适配，保证任何设备上的可用性。

* **⚡️ 零依赖**：完全使用原生 HTML5, CSS3, JavaScript 编写，无任何框架（No Framework），体积小，加载快。

## 🛠️ 收录工具 (Tools)

### 1. 卷纸计算工具 (Roll Paper Calculator)

* **路径**: `/biao/o/lc`

* **功能**: 专为工业场景设计。支持根据净重、克重计算总长度；支持分切直径计算及合并直径计算。

* **特点**: 自动保存输入数据（防丢失），结果实时计算，可视化的数据展示。

## 🎨 设计理念 (Design Philosophy)

我们相信工具应该是安静的。

* **色彩**: 使用 `#0d47a1` (深蓝) 作为主色，传达专业与冷静；背景色使用 `#f5f7fa` 降低眼部疲劳。

* **交互**: 所有的卡片和按钮都拥有微小的悬停反馈，让界面具有“呼吸感”。

* **容错**: 404 页面采用绝对路径引导，确保用户在任何错误路径下都能安全返回首页。
